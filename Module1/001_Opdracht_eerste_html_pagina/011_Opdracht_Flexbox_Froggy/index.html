CSS flexbox is een methode die je kunt gebruiken om je elementen op de juiste plek op je pagina te krijgen. Bij Flexbox moet je kiezen of je je elementen horizontaal (in rows) of verticaal (in columns) wil hebben. Het is dus geschikt als je maar 1 dimensie nodig hebt, maar niet als je op je pagina gebruik wilt maken van zowel rows als columns voor de positionering van je elementen.
